## introcode

## apitesting
Intro to how to test Python Flask API with Unit Testing and Pytest.


How to Install:
`pip install pytest`


For API testing with Flask:
`pip install pytest flask`

Or with FastAPI:
`pip install pytest fastapi httpx`

## fastapi_withdbconnection
Shows how to use Python FastAPI pydantic to connect to a locahost postgres database.
